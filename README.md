# Algorithmic Trading 

These projects reflect my interest in the intersection of statistics, financial theory, and programming. Each project explores a different strategy archetype, with a focus on rigorous backtesting, honest performance evaluation, and understanding ***why*** strategies work or fail, not just whether they do.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [SMA Crossover](./backtesting/01_sma_crossover_backtest) | Event-driven backtester, 20/200 SMA crossover on SPY |
| 2 | [Momentum Strategy](./backtesting/02_momentum_strategy_backtest) | Cross-sectional 12-1 momentum on S&P 500 with transaction costs |
| 3 | [Pairs Trading](./backtesting/03_pairs_trading_backtest) | Statistical arbitrage on cointegrated consumer staples pairs |

## Stack
python, pandas, numpy, matplotlib, yfinance, statsmodels
